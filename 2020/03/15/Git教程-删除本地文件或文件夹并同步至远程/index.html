Git 本地数据管理，大概可以分为三个区：

工作区（Working Directory）：是可以直接编辑的地方。

暂存区（Stage/Index）：数据暂时存放的区域。

版本库（commit History）：存放已经提交的数据。

工作区的文件 git add 后到暂存区，暂存区的文件 git commit 后到版本库。
